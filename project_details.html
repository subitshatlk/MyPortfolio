Suicidal Ideation Detection

The Suicidal Ideation Detection WebApp is an AI-driven application designed to identify and classify tweets that indicate suicidal ideation. This project leverages advanced Natural Language Processing (NLP) techniques and transformer-based deep learning models, such as ColBERT, to achieve high accuracy in detecting mental health-related content.

For project phase 1, a transformer-based deep learning model called ColBERT was used for classifying suicidal tweets, achieving a 94% accuracy rate against benchmark models such as SVM and GloVe.

• Leveraged CUDA to accelerate deep learning training processes.

• Phase 2 extended the application's capabilities, integrating advanced DL models such as XLNet, RoBERTa, and VGG-19 for text and image classification into a web application with Flask as the chosen framework.

• The web application was built using the Flask framework and integrates real-time data analysis through REST APIs to fetch and process live tweets. o ensure scalability and deployment efficiency, the app was containerized using Docker and optimized for performance.

Training a Continuous Bag of Words (CBoW) Model

This project focuses on training a Continuous Bag of Words (CBoW) model to generate word embeddings, following the method introduced by Mikolov et al. (2013).

The implementation involved defining a vocabulary, tokenizing the text, and converting words into one-hot encoded vectors. The model architecture includes a projection layer that maps context words into dense embeddings and a classifier layer that predicts the target word from these embeddings. Training was conducted using Cross Entropy Loss, with various learning rates tested to optimize performance. The generated embeddings were evaluated on tasks like word similarity and analogy resolution, showcasing the model's ability to capture semantic relationships.

PyTorch Python CBOW

SmartDoorLock System

The Smart Door Lock System is a two-level security solution that integrates fingerprint and facial recognition to provide secure, automated door access. Built using a Raspberry Pi, this system ensures that only authorized users can unlock the door by successfully matching both fingerprint and facial inputs. When both inputs match with a pre-registered user, the door unlocks and a green LED lights up as a visual confirmation. If either input is unauthorized, the door remains locked, and a red LED alerts to the failed authentication attempt. This system is ideal for enhancing security in homes, workplaces, bank lockers, and other sensitive areas.

Hardware Components

he system employs various hardware components including a Raspberry Pi, fingerprint scanner, Pi camera, relay module, and LEDs. For software, it utilizes Python and OpenCV for real-time face detection, leveraging the Haar classifier to detect faces. The system is designed to handle multiple cases: unauthorized fingerprints trigger an alert and keep the door locked, authorized fingerprints paired with unauthorized face recognition also result in denial of access, while only the correct combination of both inputs results in unlocking the door. This comprehensive security solution demonstrates the integration of hardware and software to provide a robust, user-friendly door locking mechanism.

Language Modeling

This project explored two key approaches to language modeling: a 4-gram model and an LSTM (Long Short-Term Memory) network. The 4-gram model was constructed using relative frequency counts and Laplace smoothing to predict the likelihood of a character based on the preceding three characters. This statistical approach provided a foundational method for sequence prediction, focusing on immediate context and avoiding zero probabilities for unseen n-grams. However, its simplicity limited its ability to capture longer-term dependencies in text.

To address these limitations, the project implemented an LSTM network. This recurrent neural network processed input sequences into character embeddings to predict the next character in a sequence, allowing it to capture more nuanced patterns and dependencies in the data. Configured with a character embedding dimension of 50 and an LSTM hidden dimension of 200, the model was trained over multiple epochs with various learning rates. The LSTM model outperformed the n-gram model in terms of perplexity, indicating a better ability to predict complex sequences. Despite the increased computational complexity, the LSTM's superior performance demonstrated the effectiveness of neural network-based models in understanding and predicting language patterns.

Food Reviews App

The Food Reviews App is a mobile application developed using Flutter and Cloud Firestore to help users find the best restaurants based on detailed reviews and cost information. Designed with user experience in mind, the app provides a platform where users can explore various dining options, check the average cost for dining, and read honest reviews posted by other customers. By aggregating this information in a user-friendly interface, the app empowers users to make informed dining choices, enhancing their overall experience.

To further enhance user interaction, a DialogFlow Chatbot was integrated into the app. This chatbot serves as a virtual assistant, efficiently addressing customer queries related to restaurant information, dining costs, and review submissions. The chatbot was designed to provide quick and accurate responses, ensuring that users can easily navigate through the app and get their questions answered in real-time. By combining the robust cross-platform capabilities of Flutter, the scalable data management of Cloud Firestore, and the intelligent conversational abilities of DialogFlow, the Food Reviews App delivers a seamless and interactive experience for users seeking the best dining options in their area.
